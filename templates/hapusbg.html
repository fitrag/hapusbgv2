Hubungi Dukungan

1. Pengenalan HapusBG

HapusBG adalah aplikasi lokal yang dirancang untuk menghapus background dari gambar secara otomatis menggunakan teknologi AI. Aplikasi ini memungkinkan Anda untuk memproses gambar secara lokal di komputer Anda tanpa perlu mengunggah gambar ke internet, sehingga menjaga privasi dan keamanan data Anda.

Cepat & Otomatis

Hapus background gambar dalam hitungan detik dengan teknologi AI canggih.

Privasi Terjaga

Semua pemrosesan dilakukan secara lokal tanpa mengirim gambar ke server eksternal.

Tanpa Internet

Bekerja tanpa koneksi internet setelah instalasi selesai.

Fitur Utama

  • Penghapusan Background Otomatis

    Deteksi dan hapus background dari gambar secara otomatis dengan akurasi tinggi.

  • Batch Processing

    Proses banyak gambar sekaligus untuk menghemat waktu.

  • Penyesuaian Background

    Ganti background dengan warna solid atau gambar kustom.

  • Format Output Beragam

    Simpan hasil dalam format PNG, JPG, atau WebP dengan transparansi.

Antarmuka HapusBG yang intuitif dan mudah digunakan

2. Persiapan

Sebelum menginstal HapusBG, pastikan sistem Anda memenuhi persyaratan minimum berikut. Persyaratan ini diperlukan untuk memastikan aplikasi berjalan dengan lancar dan optimal.

Penting: HapusBG menggunakan model AI yang membutuhkan sumber daya komputasi yang cukup. Pastikan komputer Anda memenuhi persyaratan minimum untuk pengalaman terbaik.

Python 3.8

HapusBG membutuhkan Python versi 3.8 atau yang lebih baru. Python adalah bahasa pemrograman yang digunakan untuk menjalankan aplikasi ini dan library AI-nya.

Cara Menginstal Python 3.8:

  1. Kunjungi situs resmi Python di python.org/downloads
  2. Unduh Python 3.8.x untuk Windows
  3. Jalankan installer yang telah diunduh
  4. Pastikan untuk mencentang opsi "Add Python to PATH" selama instalasi
  5. Klik "Install Now" untuk melanjutkan instalasi

Verifikasi Instalasi:

Setelah instalasi selesai, buka Command Prompt dan jalankan perintah berikut untuk memverifikasi bahwa Python telah terinstal dengan benar:

python --version

Output yang diharapkan harus menampilkan versi Python yang terinstal, misalnya: Python 3.8.10

Windows 10/11

HapusBG saat ini hanya mendukung sistem operasi Windows 10 dan Windows 11. Dukungan untuk sistem operasi lain sedang dalam pengembangan.

Persyaratan Sistem Minimum:

  • Sistem Operasi: Windows 10 (64-bit) atau Windows 11
  • Prosesor: Intel Core i3 / AMD Ryzen 3 atau yang lebih baik
  • RAM: Minimal 4GB (8GB direkomendasikan)
  • Ruang Disk: 2GB ruang kosong
  • Kartu Grafis: Tidak wajib, tetapi dapat mempercepat pemrosesan jika tersedia

Catatan: Untuk memeriksa versi Windows Anda, tekan tombol Win + R, ketik winver dan tekan Enter.

  2. 2

    Tunggu Browser Terbuka

    Selesai!

Verifikasi Instalasi

Untuk memastikan bahwa HapusBG telah terinstal dengan benar, ikuti langkah-langkah verifikasi berikut:

1. Jalankan Aplikasi

Buka aplikasi HapusBG dari menu Start atau dari shortcut di desktop. Aplikasi seharusnya terbuka tanpa error.

Jika aplikasi tidak terbuka, coba jalankan sebagai administrator.

2. Uji Fungsi Dasar

Coba unggah gambar uji untuk memastikan fungsi penghapusan background berjalan dengan baik.

  1. Klik tombol "Pilih Gambar" atau seret gambar ke area drop
  2. Tunggu proses penghapusan background selesai
  3. Pastikan hasil tampil dengan benar dan background telah dihapus

3. Periksa Integrasi Explorer

Jika Anda menginstal integrasi Explorer, periksa apakah menu konteks berfungsi dengan baik.

  1. Buka File Explorer
  2. Klik kanan pada file gambar
  3. Pastikan opsi "Hapus Background dengan HapusBG" muncul di menu
  4. Klik opsi tersebut untuk memastikan fungsinya berjalan
Pemecahan Masalah

Jika Anda mengalami masalah saat verifikasi, pastikan semua persyaratan sistem terpenuhi. Untuk bantuan lebih lanjut, lihat bagian Pemecahan Masalah atau hubungi Tim Dukungan.
